## 🚀 Main project

### Maria — Local AI Character Engine

Maria is a local AI character project built around:

- persona and behavior rules
- memory systems
- affection / trust / mood mechanics
- romance and relationship dynamics
- opinion evolution
- local LLM integration through Ollama
- API layer for future Unity / Live2D integration
- regression tests for behavior stability

The goal is not just to create a chatbot, but a more realistic character system with presence, personality and long-term interaction.
